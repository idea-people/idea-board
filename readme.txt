== Description ==

This plugin helps you to add a simple forum to WordPress.
It is currently in beta deployment progress.
Using the WordPress custom post type designed to enable a variety of plug-compatible.

- [DEMO PREVIEW](http://idea-board.ideapeople.co.kr/demo/)

* [Disqus Comment System](https://wordpress.org/plugins/disqus-comment-system) You can comment on it using.
* [WordPress Popular Posts](https://ko.wordpress.org/plugins/wordpress-popular-posts) , [Recent Posts Widget Extended](https://wordpress.org/plugins/recent-posts-widget-extended/) You can make use of the latest articles widget
* [Captcha by BestWebSoft](https://wordpress.org/plugins/captcha/) Using the plug-in can be connected to the CAPTCHA feature.
* [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/) Using the plugin, you can add users freely field.
* Non-members can also write.
* You can create a Secret.
* You can add a skin.
* Read article, article list view, write, you can set file permissions download.
* Non-members can also add an image to a post.
* You can set the number of file uploads.

= Category =

- You can choose from the WordPress editor and a text editor. The editor will be added.

- Behave more like wordpress comment.

- List, reading, writing, reply, delete, download your files, you can notice by writing privileges.
